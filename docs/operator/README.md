## 🚀 Quick Start

```bash
# Using Docker Compose
docker-compose up -d

# Using Kubernetes
kubectl apply -f k8s/
```

## 📊 Infrastructure Requirements

- **CPU**: 2+ cores
- **RAM**: 4GB+ recommended
- **Storage**: 20GB+ for database
- **PostgreSQL**: 15+
- **Network**: HTTPS enabled
